# FUTURE_UX_01 – Website Redesign

## Project Overview
This project is part of the UI/UX Design Internship with Future Interns.  
The task involved redesigning a local service business website with a strong focus on improving user experience, visual hierarchy, and conversion optimization.

For this project, I redesigned the website of Conditioned Hairdressing, a premium hair salon located in Midrand, South Africa.

The salon focuses on delivering high-quality haircare services including haircuts, colouring, smoothing treatments, and luxury hair treatments using professional products. The business prioritizes healthy hair and expert styling services. 

## Original Website
Original Website:
https://www.conditioned.co.za/

Conditioned Hairdressing is a full-service hair salon that offers services such as hair colouring, haircuts, treatments, keratin smoothing and styling services designed to keep hair healthy, manageable and vibrant.

## Design Goals
The main goal of the redesign was to improve:

• User experience and navigation  
• Website clarity and messaging  
• Visual hierarchy and layout structure  
• Conversion through stronger call-to-action sections  
• Mobile responsiveness  

## Pages Designed
The redesigned website includes the following pages:

- Homepage
- Services Page
- Product / Brand Showcase
- Contact / Booking Page

## Key UX Improvements
Some improvements implemented in the redesign include:

- Clear hero section explaining the salon’s value proposition
- Strong call-to-action buttons encouraging users to book appointments
- Improved service section layout for easier navigation
- Elegant visual design that reflects a premium salon environment
- Product brand highlights for L’Oréal, Kérastase and Kevin Murphy
- Simplified contact and booking flow

## Tools Used
- Canva (UI Design)
- UX Design Principles
- Wireframing and layout planning
